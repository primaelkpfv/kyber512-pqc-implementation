🔢 Kyber Expliqué

Basé sur LWE (Learning With Errors)

Kyber exploite la difficulté de distinguer des échantillons bruités d'une distribution aléatoire sur les réseaux euclidiens (lattices).

3 Étapes

1. Génération de clés

A ← matrice publique aléatoire

s, e ← vecteurs secrets avec peu de bruit

t = A·s + e (clé publique)

2. Encapsulation

Génère un message aléatoire de 32 bytes

Chiffre le message avec la clé publique

Retourne (ciphertext, shared_secret)

3. Décapsulation

Utilise la clé secrète pour récupérer le message

Régénère le même shared_secret

Sécurité IND-CCA2 (Indistinguishability)
